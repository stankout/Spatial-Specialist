# SEO Guide

The app includes locale-aware metadata, canonical URLs, `hreflang`, sitemap, robots rules, and static content routes. Add JSON-LD only when the matching facts are publicly visible and verified.

- Never add `AggregateRating` without genuine displayed reviews.
- Never invent local market statistics.
- Location pages require unique, useful content—not mass-generated doorway pages.
- Update `NEXT_PUBLIC_APP_URL` before launch so canonical and sitemap URLs use the real domain.
- Keep English and Vietnamese meaning aligned while allowing natural, human translation.
