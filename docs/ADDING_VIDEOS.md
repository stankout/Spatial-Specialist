# Adding Videos

The video workflow is: social video → website page → transcript → expanded guide → related CTA.

For each video provide title, slug, description, locale, publish date, platform, source URL, thumbnail, transcript, service line, related content, and CTA. Use a local licensed thumbnail or the platform's permitted thumbnail. Render a click-to-load embed so third-party iframe JavaScript is not loaded on initial page view.

No owner video URLs are currently configured; the public library intentionally displays an honest empty state.
